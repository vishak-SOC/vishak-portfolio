A modern, high-performance portfolio website built for a Blue Team defender. Designed to showcase investigations, SIEM expertise, and digital forensics investigations. Built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- **Cybersecurity Focused**: Custom "SOC Status Bar", threat intelligence feeds, and investigation showcases.
- **Modern Tech Stack**: Built with Next.js 15 (App Router), React 19, TypeScript, and Tailwind CSS 4.
- **Live Threat Intel**: 
  - Real-time security news feed from *The Hacker News*.
  - Status indicators and "System Online" visual cues.
- **Responsive & Fast**: 
  - Fully responsive across mobile, tablet, and desktop.
  - Image optimization with WebP & Skeleton loaders.
  - Framer Motion animations for a "terminal/cyber" aesthetic.
- **SEO Optimized**: 
  - Enhanced metadata for SOC/Analyst roles.
  - JSON-LD structured data for better Google indexing.
  - OpenGraph tags for social sharing.
- **Type Safety**: Strict TypeScript for robust code quality.

## Sections

1. **Hero** - "Threat Hunter" introduction with a command-center aesthetic.
2. **Skills** - Bento grid layout showcasing SIEM (Splunk), Forensics, and Network Analysis tools.
3. **Experience** - Animated timeline of security roles and freelance work.
4. **Investigations** - Showcase of Writeups, TryHackMe rooms, and GitHub projects.
5. **Certifications** - Display of security credentials.
6. **Intel Feed** - Live RSS feed tracking the latest vulnerabilities and breaches.
7. **Contact** - Secure communication channels.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
